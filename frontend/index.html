🤖 Pipeline Methods & Tools

Reinforcement Learning · S7
DQN — Deep Q-Network
Off-policy value-based RL agent. Discrete action space (6 hyper-parameters × 3 magnitudes = 18 actions). Trains a 2-layer MLP Q-function via Bellman targets from a 500-step experience replay buffer. ε-greedy exploration (ε = 0.25). Runs independently per strategy.
Sequence Model · S3
BiLSTM — Bidirectional Multi-Layer LSTM
2-layer Bidirectional LSTM used as a fixed-weight temporal smoother on JSD + entropy + token-overlap feature triples. Forward and backward hidden states are concatenated and projected to a scalar boundary score via a learned projection W_p.
Entropy Metric · S3
JSD — Jensen-Shannon Divergence
Symmetric, bounded [0,1] divergence between consecutive chunk token distributions. High JSD = strong topic shift = boundary candidate. Combined with BiLSTM: signal = 0.55 × JSD + 0.45 × LSTM.
Boundary Decision · S4
Weighted Boundary Scorer
Composite score: 25% lexical (n-gram), 20% syntactic, 15% token-type, 20% structural continuity, 20% semantic + multi-scale. Chunks with score > τ_sem are merged with neighbours if combined size ≤ 1.5×N_max.
Knowledge Graph · S5
Entity Graph + KG Store
Named entity extraction (spaCy / regex NER), co-occurrence edges, typed relation edges (SRO), persistent KG store for cross-document warm-start. Graph-weighted embedding aggregation in S6.
Embedding · S6
Ensemble Sentence Transformers
Up to 3 sentence-transformer models projected to a shared 256-dim space and averaged. Falls back to BoW if no model is available. Domain-aware context headers prepended per chunk.

📐 Score Formulas

Chunk Score (composite quality)
chunk_score = 0.4 × (1 − boundary_score) + 0.3 × ICC + 0.3 × min(JSD × 2, 1)
boundary_score ↓ = clearer boundary  ·  ICC ↑ = higher intra-chunk cohesion  ·  JSD ↑ = stronger topic shift from neighbour
JSD — Jensen-Shannon Divergence
JSD(P,Q) = ½ KL(P ∥ M) + ½ KL(Q ∥ M), M = (P+Q)/2, KL(P∥Q) = Σ P(x)·log(P(x)/Q(x))
P, Q = token unigram distributions of consecutive chunks. Symmetric & bounded in [0,1].
BiLSTM Combined Signal
signal = 0.55 × JSD_score + 0.45 × LSTM_score
signal < τ_low → merge · signal > τ_high → hard boundary · otherwise → soft boundary
RL Reward (multi-objective)
R = 0.35·quality + 0.30·coverage + 0.20·consistency + 0.15·efficiency
quality = mean(1 − boundary_score)  ·  coverage = recall proxy on probe sentences  ·  consistency = 1 − σ(sizes)/μ(sizes)  ·  efficiency = 1 − |N − N_target|/N_target
ICC — Intra-Chunk Cohesion
ICC = mean( |s_i ∩ s_{i+1}| / |s_i ∪ s_{i+1}| ) for adjacent sentence-pairs in chunk
Lexical Jaccard overlap between consecutive sentences. Higher ICC → more internally coherent chunk.
Boundary Score (S4 weighted)
B = 0.25·lexical + 0.20·syntactic + 0.15·token_type + 0.20·structural + 0.20·½(semantic+multiscale)
High B → chunks are similar → merge. Merge condition: B > τ_sem AND combined word count ≤ 1.5×N_max.
DQN Bellman Update
target[a] = r + γ·max_{a'} Q(s',a'), loss = (Q(s,a) − target[a])², Q(s,a) = W₂·tanh(W₁s+b₁)+b₂
γ=0.9 · lr=0.02 · batch=16 from 500-step replay · ε=0.25 ε-greedy exploration
